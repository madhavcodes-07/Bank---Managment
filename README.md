# 🏦 Bank Management System

A fully functional **Bank Management System** built using **Python** and **Streamlit** — 
designed to simulate real-world banking operations through a clean and interactive web interface.

## ✨ Features

- 🆕 **Create Account** — Register with name, age, email & secure PIN
- 💰 **Deposit Money** — Add funds to your account instantly
- 💸 **Withdraw Money** — Withdraw with balance validation
- 👤 **Show Account Details** — View your complete account info
- ✏️ **Update Info** — Change name, email or PIN anytime
- 🗑️ **Delete Account** — Permanently remove your account

## 🛠️ Tech Stack

- **Frontend:** Streamlit
- **Backend:** Python (OOP)
- **Database:** JSON (lightweight local storage)

## 🚀 How to Run

1. Clone the repository
   git clone https://github.com/your-username/Bank-Management-System.git

2. Install dependencies
   pip install streamlit

3. Run the app
   python -m streamlit run app.py

4. Open browser at http://localhost:8501

## 📁 Project Structure

- app.py      → Streamlit UI
- hello.py    → Bank logic (OOP Class)
- data.json   → Local database
